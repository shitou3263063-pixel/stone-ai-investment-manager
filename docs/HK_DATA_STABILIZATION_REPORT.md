## 稳定性保护

- AKShare港股数据沿用硬超时、最多2次重试、JSON缓存、字段/日期/单位验证和按接口熔断。
- HKMA新增最多2次重试、短退避、按数据集熔断、缓存回退及错误分类。
- 网络代理失败标记`NETWORK_PROXY_ERROR`；超时标记`NETWORK_TIMEOUT`；SSL、空响应和结构异常分别记录。
- 仅当主源和备用源属于同一交易日、币种一致且主源未过期时才比较价格。
- 同日价差超过阈值标记`SOURCE_CONFLICT`，禁止相关数据进入评分并降低DQS。
- 本次03033的AKShare日期为2026-07-14，现有yfinance记录为2026-07-13，系统正确标记`not_comparable`，没有制造虚假冲突。

## 完整度和决策限制

- A股分析完整度：90%，未受本轮影响。
- 港股分析完整度：65%，与实施前相同。新增两项历史行情仅用于展示和核验，HKMA两项数据已过期，因此没有人为抬高完整度。
- DQS：65，与实施前相同。
- 高置信度加仓：继续限制；当前仅允许方向性建议。
- 港股评分当前仅使用新鲜且已通过校验的数据。过期HIBOR、过期汇率以及仅展示的历史行情均未进入Opportunity Score。
